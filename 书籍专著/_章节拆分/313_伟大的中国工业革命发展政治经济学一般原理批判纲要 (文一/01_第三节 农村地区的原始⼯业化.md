## 第三节 农村地区的原始⼯业化

1978年改⾰后，在中国的早期发展阶段⼀个很明显的现象，是所谓的乡镇企业在中国⼴⼤农村如⾬后春笋般的出现。 [25] 乡镇企业的繁荣主要是因为：（1）农⺠希望有新的⽅式赚钱，或是补贴⾃给⾃⾜的农业收⼊；（2）当地村镇政府也希望找到快速发展当地经济的⽅法，以帮助农⺠致富，逃离贫困和⻢尔萨斯陷阱（这也是邓⼩平政府的要求）。

虽然，如⾬后春笋般涌现的集体所有制的乡镇企业曾被国内外众多经济学家注意到，但是这个现象与⻄⽅⼯业化历史的关系以及它在引爆中国⼯业⾰命中的伟⼤经济学意义并没有被充分理解。 [26] 它通常被认为不过是中国特有的计划经济政治体制下所产⽣的⼀个特殊（甚⾄扭曲）的向市场经济转型的现象。但是从历史的视⾓看，这种“中国式”的农村⼯业化实质上与英国⼯业⾰命前⼣发⽣在17〜18世纪的欧洲⼤陆，尤其是英国的“原始⼯业化”（proto-industrialization）浪潮⼀脉相承，具有相同的意义、机制和规律。 [27]

在整个英国⼯业⾰命前⼣的17世纪（1600—1700年）和18世纪上半叶，乡村⼯业曾在英国蓬勃兴起，繁荣了⼀个半世纪以上。在这个过程中，随着英国国内统⼀市场的建⽴，欧洲⼤陆市场和全球贸易市场的深化，越来越多的英国农⺠，包括妇⼥和孩⼦，参与到⼿⼯制造业（尤其是纺织品）⽣产中，越来越多的家庭式制造业承包和乡村作坊不断涌现，并演变成为以原始劳动分⼯和产业链聚集为特征、以远距离贸易为⽬的的原始制造业。随着⼀个半世纪的市场发酵和原始制造业发育，当批量的规模⽣产成为商⼈和早期资本家在国内外市场份额中赢得⼤规模竞争的重要⽅式时，这些“亦农亦⼯”的兼职农⺠⼯⼈和村镇作坊⽣产模式最终演变为全职⼯⼈和规模化⼤⼯⼚⽣产模式。[28]

这种根基于农村的原始⼯业化是引发⼯业⾰命所必需的程序，因为以规模化⽣产为特征的⼯业⾰命，哪怕是符合“⽐较优势”的劳动密集型产业，也需要⼀个深⼊和⼤型的市场来使得进⼀步的劳动分⼯和⼤型⼯业组织有利可图；这反过来需要依靠草根群体的充⾜的⾼收⼊（⼯资）和购买⼒，需要在不破坏粮⾷安全的情况下，吸引⼤量的“以⾷为天”的⾃给⾃⾜的分散的农⺠个体投⼊到以分⼯协作为特征的制造业⽣产中。因此，在开始阶段，⼤⾯积使⽤农村剩余劳动⼒和农⺠的空闲时间来“就地”⽣产原始的低附加价值劳动密集型原⽣⼯业品，并通过远距离贸易来赚钱，是⼀种⼗分经济、有效和⾃然的⽅式；这是发酵市场、培育企业家、发展供应链和商业配送⽹络、提⾼农村和城市商业化对⽇常制造品的需求和⽣产⼒、提⾼农⺠收⼊、产⽣地⽅政府收⼊⽤于本地基础设施建设，并最终启动⼯业⾰命的诀窍。 [29]

中国的乡镇企业繁荣与早期欧洲的原始⼯业化之间的关键区别在于，当年的欧洲和英国主要靠有实⼒的商⼈阶层牵头在农村投资、承包、组织村镇作坊⼯业⽣产和其产品的远距离销售：他们雇佣农⺠的闲暇时间和剩余劳⼒并为之提供原材料，来“就地”⽣产满⾜远距离市场需求的制造业产品（主要是纺织品） [30]，然后把制成品分销到全国和世界各地的商业⽹络。这种“家庭包⼯制”（putting-out system）在⼯业⾰命前的欧洲⾮常普遍。 [31] 因此在欧洲和英国，⼴⼤的商⼈阶级是⼤市场的创造者和⽣产的组织者，是原始⼯业发育的“酵⺟”和催化剂。然⽽在20世纪七⼋⼗年代的中国，由于缺乏这样⼀个庞⼤的商⼈阶级来开辟国内和国际市场，创造远距离市场和组织原始⼯业⽣产的重任主要是由中央、地⽅和村镇⼀级的政府承担（这⼀点还将在后⾯详细分析）。

中国式的农村（原始）⼯业化，通过1978年后⼤量乡镇企业（集体所有）的涌现，终于结束了计划经济时代由中央计划和重⼯业为主政策导致的“短缺经济”现象。1978年改⾰后的5年内，中国就成功地结束了⼴泛存在于⾷品、⾁类、纺织品和其他轻⼯业消费品的配额制度。

但是，更为重要的是，中国这场农村⼯业化浪潮和乡镇企业繁荣也终于在20世纪80年代末和90年代初引爆了中国近代史上苦苦等待和久盼的第⼀次⼯业⾰命。第⼀次⼯业⾰命的基本特征就是基于劳动分⼯原理的规模化⽣产，或劳动密集型“⼯⼚体制”在全国范围内的普遍建⽴。但是，由于80年代的中国依旧贫穷，⼤部分⼈⼝仍然在农村，⽽且早期的乡镇企业依赖的是原始⽣产技术，因此很少有⼈（可能除了中国改⾰开放的总设计师邓⼩平外）能够敏锐抓住或意识到这场农村⼯业化浪潮的深远历史意义和经济学意义。事实上，经过10年的原始⼯业“发酵”和市场孕育，1988年前后的中国已经处在引爆⼯业⾰命的门⼝，却丝毫没有引起国内外经济学家和媒体的重视。考虑到当年英国⼯业⾰命的前夜和最初阶段也并不是⼈们后来所想象的那么戏剧性和引⼈注⽬，这种敏锐意识的缺乏也并不奇怪。因为即使是像亚当·斯密、⻢尔萨斯、⼤卫·李嘉图、约翰·穆勒这样的天才经济学家，也完全没有意识到在18世纪末和19世纪初的英国，⼀场⼈类历史上史⽆前例的第⼀次⼯业⾰命正在他们眼前展开。⽽这场⼯业⾰命恰好是由英国从17世纪（或更早）到18世纪中叶的农村⼯业化运动所引爆。直到第⼀次⼯业⾰命结束和第⼆次⼯业⾰命开始后（即1840-1860年代左右），这场由农村原始⼯业化引爆的英国⼯业⾰命才被如⻢克思和恩格斯这样少数具有洞察⼒的政治经济学家所察觉和意识到。

因此，遵循英国⼯业⾰命同样的“历史逻辑”，中国的第⼀次⼯业⾰命也发源于乡村，即20世纪70年代末和80年代发展起来的乡镇企业虽然这些企业在所有制上与当年英国的乡村作坊完全不同（这正是⿇痹和误导经济学家们的地⽅）。在中央⼀系列农村政策的指导下和各级地⽅政府的⼤⼒帮助和扶持下，这些乡镇企业由未受教育的农⺠构成、组织、管理。这些农⺠与他们17〜18世纪的清朝祖先没什么两样（可能除了没有辫⼦以外）。有些经济学家和经济史学家过分强调正规科学知识（所谓⼈⼒资本）在⼯业⾰命中的关键作⽤，把中国早年没有成功在17〜18世纪发展出英国式的纺织机和蒸汽机归因于⽂化⽔平低下和缺乏现代教育的中国农⺠。但是，事实上也正是这样⼀些农⺠在20世纪末亲⼿点燃了中国的⼯业⾰命。 [32]

中国原始⼯业化的速度与⻄⽅发展史相⽐却是史⽆前例的。1978—1988年⼗年间，中国的乡镇企业从150万个变为1890万个，增⻓了12倍；农村⼯业总产值从515亿元（占GDP的14%）增加到7020亿元（占GDP的46%），增⻓超过13.5倍；农村⼯业的就业从2800万⼈发展到9500万⼈，增⻓超过3倍；农⺠总收⼊从87亿元到963亿元，增⻓12倍；村镇企业总资本存量从230亿元到2100亿元，增⻓超过9倍。同时，农村⼯⼈作为总农村劳动⼒的⼀部分，从9%增⻓到23%。 [33]

这种爆炸式的增⻓在进⼊20世纪90年代以后在中国农村继续发⽣，像引爆了热核反应堆⼀样，⼀个爆炸导致更多的爆炸，⼀个增⻓导致更多的增⻓。到了2000年，乡镇企业职⼯超过了1.28亿⼈（不包括去城⾥打⼯的农⺠⼯），占到了中国农村总劳动⼒的30%。农村⼯业总产值达到11.6万亿元，是1988年的16.5倍，1978年的225倍。在1978—2000年，村镇⼯业产量平均增⻓率为每年28%，即23年⾥平均每三年就翻⼀倍。即使调整通货膨胀因素后 [34]，真实的增⻓率也会在21%左右，为中国全国真实GDP增⻓率的两倍（即每3.7年增⼀倍），意味着村镇实际总产量在1978—2000年间增⻓了⾄少66倍。这种“⾃我复制”和核裂变式的规模和速度在如此⻓时间内维持，在⼈类历史上还没有记录过。亚当·斯密没⻅过，⻢克思也从未听说过甚⾄想象过。它使英国原始⼯业化阶段和⼯业⾰命相形⻅绌。

经过1978—1988年⼗年期的乡村原始⼯业的⼤规模繁殖和爆发式增⻓，罗斯托（1960）称作“经济起⻜”的时刻注定会到来。第⼀个时刻发⽣在1988—1998年的中国第⼀次⼯业⾰命期间。这个期间，基于劳动分⼯原理的劳动密集型规模化轻⼯企业在农村和城市⼤规模兴起，与发⽣在1760—1830年的英国第⼀次⼯业⾰命类似，但是以更加猛烈惊⼈的速度和威⼒爆发。 [35] 第⼆个关键时刻⼤概在1995—2000年，当时中国⼈均GDP达到了1000美元（以2000年的美元计价）。但是，离开对产业结构的分析，⼈均GDP 1000美元也许只是个随意和空洞的数字，实际意义不⼤，虽然世界银⾏经常把它当作⼀个进⼊中等收⼊国家俱乐部的临界值。许多拉丁美洲国家⼈均收⼊是这个收⼊的⼏倍甚⾄超过⼗倍，却缺乏进⼀步增⻓的动⼒。1995—2000年对中国真正重要的，是批量⽣产劳动密集型轻⼯业品的能⼒，以及“中国制造”在国内外市场上的需求已经变得⾮常⼤——如此之⼤，以⾄于中国对能源、动⼒机械、基础设施和巨型机器设备的需求（购买⼒）⼤到在中国⾃⼰批量⽣产这些重⼯业品变得⾮常有利可图。

因此，在1995—2000年，中国已经处在引爆第⼆次⼯业⾰命的门⼝。所谓第⼆次⼯业⾰命，就是采⽤规模化⽣产⽅式来⽣产那些造就了（并维持）第⼀次⼯业⾰命⽣产⽅式所需要的（规模化）⽣产⼯具和中间产品本⾝。或简单地说，就是⽤机器来批量⽣产机器。正如经济史学家兰德斯注意到的，“纺纱和织布是⼀回事，能够⽤机器来纺纱织布⼜是另⼀回事。”（David Landes，1999，p380）对此我们得加上⼀句：⽤机器来纺纱织布是⼀回事，能够⽤机器来批量⽣产这些机器（纺纱机和织布机）更是另⼀回事！如果发明机器来纺纱织布是第⼀次⼯业⾰命的特征，那么发明机器来批量⽣产这些⽣产⼯具和运输⼯具以及所有中间产品则是第⼆次⼯业⾰命的特征。

尤其是，中国加⼊世界贸易组织之后的21世纪早期，中国正式进⼊重⼯业建设阶段（由巨⼤的国内储蓄和国际纺织品市场需求以及其他的轻⼯业品市场⽀持），开启了化学品、⽔泥、电、钢铁、⾦属品、内燃机、卡⻋、汽⻋、轮船、⾼速公路、铁路、⾼速铁路、农业机械、纺织机器等等，以及⽤来⽣产各种轻⼯业品如电脑、冰箱、摩托⻋、电视、洗⾐机、家具等等的组装线和机床的⼤批量⽣产。

⼀旦按正确顺序引爆第⼀次和第⼆次⼯业⾰命，其⾃我繁殖和盈利的能⼒是任何⾦融危机都难以阻⽌的。1997年的亚洲⾦融危机阻⽌不了中国的发展，2007年的全球⾦融危机（它使中国出⼝总量与其增⻓趋势相⽐永久性地减少了近40% [36] ）也没能阻⽌中国的发展。因此，只要⼀个经济体完成了原始⼯业化的胚胎发育和第⼀次⼯业⾰命所带来的健康“婴⼉”时期的成⻓，即便在危机中也会发展，这与清朝晚期50年的变法图强、⺠国时期40年的⻛⾬飘摇和计划经济时期30年的⼤起⼤落与最终萧条形成鲜明对照。就如19世纪的美国：经历了15次⾦融危机和⼀次四年的国内战争，这些都没有阻⽌美国强劲发展成为下⼀个世界⽣产中⼼和超级⼤国。这不是因为美国的政治体制（如制度经济学家所吹嘘的），⽽是因为美国重商主义政府⾃建国伊始就采纳了⼀个正确的、有序推进的、从原始⼯业化到轻⼯业化再到重⼯业化的发展战略（详⻅第四章）。

⽤现代⻄⽅经济学的术语将中国在20世纪80年代和90年代的发展经历模型化（即肤浅地数学化）的⽅式就是想象两个不同的⽣产技术： [37] （1）原始⼿⼯作坊技术，其⽣产函数是 $\cdot y = n ^ { \textnormal { 0 . 5 } }$ ；（2）采⽤机器的⼯⼚技术，其⽣产函数 $y = 2 n - 1 0 0 \mathfrak { c }$ 。y 表⽰为产量（⽐如纺织品），n 为劳动投⼊。假定⽣产的⽬的都是为了交换，⽽不是⾃给⾃⾜。第⼆个技术涉及固定⽣产成本100，它反映采纳规模化⽣产所需要付出的在固定资本投⼊、创新、⼯业组织等⽅⾯的固定成本。但第⼀个技术没有这样的固定成本（或可以忽略不计）。

第⼀个技术是劳动报酬边际递减的⼿⼯作坊产业，第⼆个是使⽤机器的规模产业。就边际产量⽽⾔，因为使⽤机器，劳动在第⼆个⽣产技术中更具规模化效应。然⽽，当市场需求很⼩时（假设市场需求y=1单位），⽤第⼀个技术更加有利可图，即只⽤⼀单位的劳动投⼊就能满⾜市场需求。第⼆个技术需要最少51个单位的劳动投⼊，才能抵扣掉100的固定成本，以实现正的利润收⼊，因此在市场很⼩时是个亏本的技术投⼊。但是，⼀旦y （纺织品）的需求或者市场规模增加，⽐如从y =1单位增加到y =10个单位，⽤第⼀个技术需要100个单位的劳动投⼊，⽽第⼆个技术只需要55个单位的劳动投⼊。

现在，想象⼀个村庄有200个家庭，每个家庭有⼀个单位的劳动⼒。如果他们都独⽴经营，这个村庄的总GDP就是200个单位。但是，如果他们组织起来成为⼀个⼯⼚，⽤第⼆个技术通过团队⼯作和劳动分⼯来共同⽣产，那么这个村庄的GDP将是300个单位。更重要的是，这个村庄的家庭越多，⽤第⼆个⽣产模式会有更多的产量和更⾼的劳动⽣产率。因此，市场规模（需求）决定劳动分⼯、劳动组织和新技术使⽤的程度。同时，请记住前⾯提到的粮⾷安全问题和纺织品需求的收⼊弹性问题：如果第⼀种⽣产技术对应粮⾷⽣产，那么即便对纺织品的市场需求很⼤，这个村⼦也不敢贸然采纳新技术，把农⺠变成⼯⼈，除⾮它能够在市场上⽤多余的纺织品随时随地换到粮⾷。

然⽽，社会（劳动）分⼯所依赖的⼤规模市场（需求）从哪⾥来？是谁⾸先创造了它？
